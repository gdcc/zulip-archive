Technical Plan

Demo collection for testing

Please note that all feedback on this feature should be in the Github issue

For the past year or so, we have been working with our developers and community to support TK labels in Dataverse.
We are finally at the stage where we can ask the broader community to test the feature on demo Dataverse and provide any feedback we can use to improve this feature. Feedback should be received no later than January 15th, 2025, as we plan to implement the feature early in the new year.

The feature exists as a metadata block in the demo environment that can be enabled at your installation or sub-collection level. It is currently utilizing a sandbox server at Local Context.

You can navigate to the demo test collection here: https://demo.dataverse.org/dataverse/LCtests/
and note there is a slight delay in the labels view, and that bug is being worked on.
