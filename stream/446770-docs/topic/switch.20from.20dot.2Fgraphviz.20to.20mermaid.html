Ash Manda (May 28 2026 at 19:43):

My two cents when it comes to hosting the docs specifically:

  1. Maintenance burden reduces. With Graphviz you always need access to a CI pipeline or a full server runner just to render the documentation
  2. DOT language is not human friendly whatsoever. Mermaid reads like English, very easy for contributors to get started with
  3. Mermaid renders natively on GitHub, DOT won't. Contributors can easily preview changes
  4. Graphviz is a lot more powerful when it comes to layout, but Mermaid has a more active ecosystem since GitLab, Notion, etc etc everything supports mermaid by default
