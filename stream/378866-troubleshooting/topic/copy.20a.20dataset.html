@Carlos Perez-Miguel no, simply copying a dataset is not supported (please feel free to create an issue about this!).

Usually we recommend that you:

  • download all the files
  • download the dataset in JSON format (Dataverse's native JSON format)
  • use the JSON to create the dataset via API
  • upload the files
