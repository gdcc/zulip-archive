Topic: Dataverse 6.10.1 release (fixes download bug in 6.10)

Philip Durbin 🚀 (Mar 26 2026 at 20:23):

As I wrote on the mailing list...

The bad news is that the Dataverse 6.10 release had quite a bad bug in it that caused single-file downloads (as opposed to multi-file zip download) to silently fail from the web interface (the API was unaffected).

The bug is bad enough that we've deleted the 6.10 war file and installer. No one should use them.

The good news is that the bug has been fixed in Dataverse 6.10.1!

We've adjusted the 6.10 upgrade instructions to use the 6.10.1 war file instead.

For details, please see the 6.10.1 release notes: https://github.com/IQSS/dataverse/releases/tag/v6.10.1
