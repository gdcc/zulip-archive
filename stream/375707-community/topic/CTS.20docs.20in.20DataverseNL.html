Laura Huis in 't Veld (Oct 31 2024 at 12:34):

Hi all! There is indeed a collection for CoreTrustSeal within the DANS collection at DataverseNL. Once the certificate is obtained, the corresponding PDF is published. The title of each dataset and the PDF contains the 'requirement years'. There are datasets with the older requirements (2020-2022) but also recent ones - 2023-2025.

The deposit is an automated process. If the workflow in the submission system has ended, the corresponding file will be published.
